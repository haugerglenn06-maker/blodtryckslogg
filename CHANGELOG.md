# Changelog

## [1.4.9] – 2025-12-18
### Added
- Export till PDF med stöd för mobil och desktop
- CSV-export
- Säkerhetskopiering till JSON-fil
- PWA-stöd (offline, installera som app)
- Visuell bekräftelse (toast) vid sparande
- Haptic feedback på mobil

### Improved
- Mobilanpassad layout
- Automatisk scroll till toppen efter sparande
- Tangentbord stängs efter inmatning
- Gruppering per dag med sorterade tider

### Fixed
- Stabilitet vid PDF-export
- Förbättrad användarupplevelse på mobil
